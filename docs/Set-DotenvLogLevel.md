# Set-DotenvLogLevel

## SYNOPSIS
Sets the log level for the Dotenv module.

## SYNTAX

```
Set-DotenvLogLevel [-Level] <LogLevel> [<CommonParameters>]
```

## DESCRIPTION
Sets the log level for the Dotenv module.

## EXAMPLES

### Example 1
```powershell
PS C:\> Set-DotenvLogLevel 'Info'
```
