# Get-DotenvLogLevel

## SYNOPSIS
Gets the log level configured for Dotenv.

## SYNTAX

```
Get-DotenvLogLevel [<CommonParameters>]
```

## DESCRIPTION
Gets the currently set log level for Dotenv.

## EXAMPLES

### Example 1
```powershell
PS C:\> Get-DotenvLogLevel
Debug
```
