# Disable-Dotenv

## SYNOPSIS
Disables the module without removing it from the session.

## SYNTAX

```
Disable-Dotenv
```

## DESCRIPTION
Disables the module without removing it from the session.
Equivalent to `$Dotenv.Enabled = $true`.

## EXAMPLES

### Example 1
```powershell
PS C:\> Disable-Dotenv
```
