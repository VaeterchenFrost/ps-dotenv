# Enable-Dotenv

## SYNOPSIS
Enables the Dotenv module.

## SYNTAX

```
Enable-Dotenv [<CommonParameters>]
```

## DESCRIPTION
Enables the Dotenv module.
This is useful because you can temporarily disable and enable the effects of this module without changing your profile.

## EXAMPLES

### Example 1
```powershell
PS C:\> Enable-Dotenv
```
