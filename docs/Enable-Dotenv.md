# Enable-Dotenv

## SYNOPSIS
Enables the module back.

## SYNTAX

```
Enable-Dotenv
```

## DESCRIPTION
Enables the module back.
Equivalent to `$Dotenv.Enabled = $true`.

## EXAMPLES

### Example 1
```powershell
PS C:\> Enable-Dotenv
```
